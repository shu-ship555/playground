# nextjs/

Next.js App Router の仕組みを TypeScript で学ぶためのサンドボックスです。

| 項目 | 内容 |
| :--- | :--- |
| **Framework** | Next.js v16（App Router） |
| **言語** | TypeScript |
| **主な内容** | ページルーティング、Server / Client Components、データフェッチ、レイアウト構造 |

## セットアップ

```bash
cd nextjs
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## ファイル構成

```
nextjs/
├── app/
│   ├── layout.tsx    # ルートレイアウト
│   ├── page.tsx      # トップページ（ここを編集して学習する）
│   └── ...
└── public/
```
