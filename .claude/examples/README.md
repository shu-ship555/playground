# 実験・サンプル集

Claude Code を使った実験結果やサンプルコードを記録します。

## 試してみたいこと

### 1. エージェントの活用
- `Explore` エージェント: コードベースの探索
- `Plan` エージェント: 実装計画の立案
- `general-purpose` エージェント: 複雑なマルチステップタスク

### 2. MCP サーバー
- Notion MCP: Notion のページ作成・編集
- WordPress MCP: 記事の作成・更新
- カスタム MCP サーバーの作り方

### 3. メモリシステム
- `~/.claude/projects/<project>/memory/` に保存される
- `MEMORY.md` がインデックスファイル
- `user`, `feedback`, `project`, `reference` の4タイプ

### 4. TodoWrite ツール
タスクを分解して進捗管理する例:

```
TodoWrite で以下を管理:
- [ ] 要件定義
- [x] 環境構築
- [ ] 実装
- [ ] テスト
```
