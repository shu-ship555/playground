description: 新しいサンドボックスディレクトリを作成し、README.md のディレクトリ構成を更新する
---

新しいサンドボックスディレクトリを作成してください。

引数: $ARGUMENTS
（例: `threejs-advanced` や `react vite で React の学習環境` のように、ディレクトリ名と概要を渡す）

以下の手順で進めてください：

1. 引数からディレクトリ名と用途を読み取る（引数がなければユーザーに確認する）
2. ルートの `README.md` を読み込み、既存のディレクトリ構成と書式を確認する
3. 新しいディレクトリを作成する
4. そのディレクトリに最小限の `README.md` を作成する（概要・技術スタック・起動方法を含む）
5. ルートの `README.md` のディレクトリ構成と「各ディレクトリの概要」セクションに新しいエントリを追記する

既存のエントリの書式・スタイルを踏襲すること。
